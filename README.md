# LookStore – Aplicación Móvil

LookStore es una aplicación móvil desarrollada en **Kotlin + Jetpack Compose**, enfocada en la venta y visualización de productos de moda.  
Incluye gestión de usuarios, carrito de compras, historial de pedidos, perfil editable, integración con API externa y conexión a microservicios en Spring Boot.

##  Funcionalidades principales

### **Aplicación móvil (Android – Jetpack Compose)**
- Registro y login de usuarios  
- Catálogo de productos  
- Detalle del producto  
- Carrito funcional  
- Realización de pedidos  
- Historial de compras  
- Perfil de usuario (edición de datos)  
- Gestión de productos (solo administrador)  
- Persistencia local con Room  
- Consumo de API externa usando Retrofit  
- Conexión al backend propio en Spring Boot  
- Uso de periféricos:
  - Selección de imagen desde galería  
  - Notificación local de bienvenida
